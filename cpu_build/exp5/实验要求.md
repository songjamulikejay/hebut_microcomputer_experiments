作业要求：模拟8086 CPU 程序实现部分指令集及扩展

一、作业目的
本次作业旨在通过编程模拟8086 CPU 的工作环境，实现并展示该 CPU 支持的所有寻址方式、数据传输指令、算术运算指令、逻辑运算指令，并扩展实现字符串命令、处理器控制类指令和程序控制类指令。通过此实践，学生将深入理解8086 CPU 的指令集架构，掌握其操作原理，并提升编程能力和对计算机底层工作机制的认识。

二、作业内容
模拟环境搭建
使用 C/C++ 或其他适合的编程语言搭建一个模拟8086 CPU 的程序框架。
实现 CPU 的基本组成部分，包括寄存器组（如 AX, BX, CX, DX, SP, BP, SI, DI, IP, CS, DS, ES, SS 等）、内存模型、指令解码器、执行单元、标志位等。

实现中断和异常处理机制（可选）。

实现所有寻址方式

在模拟程序中实现8086 CPU 支持的所有寻址方式，包括立即数寻址、寄存器寻址、直接寻址、寄存器间接寻址、基址加变址寻址、相对寻址等。

为每种寻址方式编写测试案例，确保它们能够正确解析和操作数据。

实现数据传输指令

实现所有基本的数据传输指令，如 MOV、PUSH、POP、XCHG 等。

编写测试代码，验证这些指令在不同寻址方式下的正确性。

实现算术运算指令

实现所有基本的算术运算指令，包括 ADD、SUB、ADC（带进位加法）、SBB（带借位减法）、INC、DEC、MUL（无符号乘法）、IMUL（有符号乘法）、DIV（无符号除法）、IDIV（有符号除法）等。

编写测试代码，验证这些指令在不同操作数和寻址方式下的运算结果和标志位设置。

实现逻辑运算指令

实现所有基本的逻辑运算指令，包括 AND、OR、XOR、TEST 等。

编写测试代码，验证这些指令在不同操作数和寻址方式下的逻辑运算结果和标志位设置。

实现字符串命令

扩展实现字符串命令，如 MOVSB、MOVSW、CMPSB、CMPSW、SCASB、SCASW、LODSB、LODSW、STOSB、STOSW 等。

编写测试代码，验证这些指令在不同操作数和寻址方式下的正确性。

实现处理器控制类指令

扩展实现处理器控制类指令，如 NOP、HLT、CLC、STC、CMC、CLD、STD、CLI、STI 等。

编写测试代码，验证这些指令对 CPU 状态和标志位的影响。

实现程序控制类指令

扩展实现程序控制类指令，如 JMP、CALL、RET、INT、IRET、LOOP、LOOPZ、LOOPNZ 等。

编写测试代码，验证这些指令在不同条件下的跳转和调用行为。

综合测试

编写一系列综合测试案例，涵盖所有指令和寻址方式的组合，确保模拟程序的全面性和准确性。

分析测试结果，调试并优化模拟程序，以提高其性能和稳定性。

三、提交要求

源代码：提交完整的模拟程序源代码，包括所有模块的实现和测试代码。

文档：提交详细的文档，说明模拟程序的设计思路、实现过程、关键代码解析、测试方法和测试结果分析。

测试报告：提交测试报告，列出所有测试案例及其执行结果，对出现的问题进行说明并给出解决方案。

四、评分标准

功能实现（50%）：模拟程序是否实现了所有指定的寻址方式、指令以及扩展的字符串命令、处理器控制类指令和程序控制类指令，功能是否完整。

代码质量（20%）：代码的可读性、可维护性、注释的详细程度以及是否遵循了良好的编程习惯。

测试覆盖（20%）：测试案例的覆盖范围和深度，是否充分验证了模拟程序的正确性。

文档与报告（10%）：文档的完整性和准确性，测试报告的详细程度和清晰度。

通过本次作业，学生将能够深入理解8086 CPU 的指令集架构，掌握其操作原理，并提升编程能力和对计算机底层工作机制的认识。
